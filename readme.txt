调试说明
1. 运行weex main.we即可
2. weex版本，运行weex --version ,输出是0.6.3。里面使用到了panmove，在调试的时候遇到警告，说这个特性将会被移除。测试环境可能要同版本
3. 此版本是有背景音乐的版本，视频放在网上，需要联网，否则会提示资源加载失败。有音乐的版本在手机上运行会出现小概率卡顿事件
4. 网页端手势位置panmove检测不灵敏，勉强能用，手机端以及playground扫码没有这个问题
5. 用playground扫码的时候因为顶部actionbar的存在，会导致整个页面下移，与apk看到的实际位置不一致（apk去掉了actionbar，位置整体上移，看起来更自然）
6. 如果听不到背景音乐可能是网络加载太慢，或者音量不够大。
7. 测试时需要联网
